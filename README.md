# DistModulation-Project
Plugin Audio in formato Vst3 di una Catena Modulare Di Effetti

Il Plugin presenta 3 effetti:

-Distorsore: 3 tipologie di distorsione (Overdrive, Lead, Fuzz)
             Diversi parametri disponibili tra cui l'accensione e lo spegnimento di un filtro HighPass posto pre-distorsione (Button HP)
             Sviluppi futuri per questo effetto -> Migliorare la qualità della distorsione

-Delay: Delay frazionario
        Il parametro di Tone modifica solo la timbrica delle ripetizioni
        Sviluppi futuri per questo effetto -> Impostazione del tempo di ritardo in bpm, permettere al parametro del Tone di agire anche su tutto il segnale,              implementazione di saturazioni e oscillatori per rendere il suono più analogico

-Chorus: Dual-Mono Chorus
         Sviluppi futuri per questo effetto -> aumentare il numero di voci rendere l'effetto più presente


I 3 effetti sono implementati in una catena modulabile che permette il posizionamento degli effetti in qualsiasi punto della catena secondo la preferenza dell'utente
Sviluppi futuri per la catena -> Rendere il posizionamento degli effetti tramite un'operazione di Drag&Drop
